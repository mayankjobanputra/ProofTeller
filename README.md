# ProofTeller
Repository for our AACL paper "ProofTeller: Exposing recency bias in LLM reasoning and its side effects on communication".

## Note:
This dataset is obfuscated on purpose (i.e., to prevent LLMs from training on it). Researchers are advised to use the script to deobfuscate the dataset first before using it.

## Explicit AI/LLM Training Restriction

For the avoidance of doubt, the "NonCommercial" and "NoDerivatives" restrictions of this license explicitly prohibit the use of this dataset, in whole or in part, for the purposes of machine learning training, LLM pre-training, fine-tuning, retrieval-augmented generation (RAG) databases, alignment, or testing by any commercial entity. 

Non-commercial, academic researchers may use the dataset to evaluate model performance (benchmarking), provided the raw or decrypted data is not permanently ingested into, or used to build, a new or derived model artifact for distribution.
